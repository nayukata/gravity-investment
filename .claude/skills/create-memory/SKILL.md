# メモリー作成スキル

CLAUDE.md および .claude/rules/ ファイルを作成・更新するためのガイダンスを提供する。

## 作成前の確認

作成前に以下を出力:

```
【メモリー作成】
種類: [CLAUDE.md / rules / CLAUDE.local.md]
パス: [ファイルパス]
トピック: [1つのトピック名]
```

## ベストプラクティス

詳細は `@references/memory-best-practices.md` を参照。

### 必須ルール

- **1ファイル1トピック**
- **箇条書き形式**（`-` で列挙）
- **具体的に記述**（「適切にフォーマット」❌ →「2スペースインデント」✓）
- **MUST/SHOULD/NEVER** で強度を明示
- **コード例は最小限**（1つで十分）

### 禁止事項

- 曖昧な指示（「ベストプラクティスに従う」）
- 1ファイルに複数トピック
- 冗長な例

## ファイル形式

### rules ファイル

```markdown
---
paths: '対象glob'
---

# トピック名

- **MUST**: 具体的ルール
- **SHOULD**: 推奨
- **NEVER**: 禁止
```

### CLAUDE.md

```markdown
# プロジェクト名

## コマンド

- Build: `pnpm build`

## ルール

@.claude/rules/
```

## 作成後の確認

- [ ] 1ファイル1トピックか?
- [ ] 箇条書き形式か?
- [ ] 具体的な指示か?
- [ ] 冗長な例がないか?
